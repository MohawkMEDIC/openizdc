OpenIZ Disconnected Client
--

This helpfile documents the OpenIZ interoperation layer which is used by OpenIZ disconnected client applications to interact
with the mobile immunization management system service. The operations included in the interoperation layer can be used for:

* Saving, Querying, Updating and Obsoleting data objects to/from the mobile immunization store.
* Controlling the application and interacting with device hardware synchronously
* Interacting with the applet manifest system including retrieving localization string and session management
* Retrieving certain application control objects such as configuration and security decisions
* Interacting with the care plan forecasting engine.
